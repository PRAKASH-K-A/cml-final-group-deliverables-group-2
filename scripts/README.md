# Scripts Layout

This folder contains operational scripts grouped by purpose.

- `build/`: Build and container run helpers (`build.sh`, `run-from-docker-hub.sh`).
- `ops/`: Day-to-day operational utilities (`start-local.ps1`, market/circuit-breaker helpers).
- `debug/`: Targeted debugging scripts.
- `tests/`: E2E and performance test runners.
- `validation/`: Validation and reporting scripts.

Run commands from repository root to keep relative paths consistent.
